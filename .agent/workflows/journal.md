## Purpose

Explore recent code changes and write concise journal entries.

## Workflow

1. Review recent git commits and changes on the current branch
2. Identify the most important events, key changes, impacts, and decisions
3. Write concise journal entries capturing the essence of the work
4. Save entries in `./docs/journals/` directory

## Notes

- Journal entries should be focused and brief
- Capture key decisions and their rationale
- Note any significant impacts or trade-offs
