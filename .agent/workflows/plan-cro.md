Create a Conversion Rate Optimization plan.

## Workflow

1. **Audit** existing content and conversion funnel
2. **Research** industry benchmarks and competitor approaches
3. **Identify** optimization opportunities ranked by impact
4. **Create** detailed plan in `./plans/` directory with phases
5. **Report** plan summary

## Report Output

Save to `plans/{YYMMDD}-{HHMM}-{slug}/`:
- `research/researcher-XX-{topic}.md` — ≤150 lines each
- `plan.md` — overview with YAML frontmatter
- `phase-XX-{name}.md` — detailed optimization phases

## Notes
- Focus on measurable conversion improvements
- Prioritize high-impact, low-effort changes
- Plan only — does not implement
