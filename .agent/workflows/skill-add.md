description: Add new reference files or scripts to a skill
---

Add resources to an existing skill.

## Arguments
- Skill name (required)
- What to add (required)

## Workflow

1. **Read** the existing skill's SKILL.md
2. **Research** the topic if needed (documentation, GitHub repos)
3. **Create** the new reference or script file
4. **Update** SKILL.md to reference the new file
5. **Test** scripts if applicable
6. **Report** what was added

## Notes
- Keep reference files under 100 lines (progressive disclosure)
- Skills are practical instructions, not documentation
- Test all scripts before delivering
