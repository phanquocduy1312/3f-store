Analyze existing content and create a CRO (Conversion Rate Optimization) plan.

## Workflow

1. **Audit** existing content (headlines, CTAs, copy, layout)
2. **Identify** conversion bottlenecks and weak points
3. **Research** industry benchmarks and best practices
4. **Create** optimization recommendations with priority ranking
5. **Write** improved copy variants for A/B testing
6. **Report** findings with before/after comparison

## Notes
- Focus on measurable conversion improvements
- Prioritize high-impact, low-effort changes first
- Include specific copy recommendations, not just guidelines
