Parse error logs, identify root cause, and fix.

## Workflow

1. **Read** log files or provided log output
2. **Identify** error patterns and root cause
3. **Trace** to source code location
4. **Implement** fix
5. **Verify** fix resolves the logged error
6. **Report** findings

## Notes
- Focus on stack traces and error messages
- Use `debugging` skill for systematic root cause analysis
