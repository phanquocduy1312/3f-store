## Purpose

Review the current branch and recent commits, providing a detailed summary of all changes.

## Workflow

1. **Check** current branch and recent commit history
2. **Analyze** all changes — what was modified, added, or removed
3. **Summarize** the overall impact and quality of changes:
   * Files changed and their purpose
   * New features or fixes introduced
   * Potential issues or concerns
   * Code quality observations

## Important Notes

* **Do not** start implementing anything — only review and report
