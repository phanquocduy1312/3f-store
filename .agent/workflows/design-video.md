Create a design inspired by a provided video reference.

## Workflow

1. **Analyze** video using `ai-multimodal` skill
2. **Extract** design elements: motion patterns, color palette, typography, layout flow
3. **Document** design guidelines from video analysis
4. **Implement** code following extracted guidelines
5. **Add** motion/animation inspired by the video
6. **Deliver** working design

## Notes
- Use `ai-multimodal` skill for video analysis
- Capture motion language and transitions from video
