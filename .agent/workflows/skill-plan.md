Create a detailed plan for a new skill before implementing.

## Workflow

1. **Clarify** requirements (ask probing questions)
2. **Research** the topic thoroughly
3. **Design** skill structure and content
4. **Create** plan in `./plans/` directory
5. **Ask** user to review and approve

## Notes
- Plan only — does not create the skill
- Consider: SKILL.md structure, references needed, scripts needed
- Challenge assumptions — explore 2-3 approaches
