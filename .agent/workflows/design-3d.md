Build 3D web experiences using Three.js.

## Workflow

1. **Understand** the 3D design requirements
2. **Activate** `threejs` skill for Three.js patterns
3. **Activate** `frontend-design` skill for aesthetic direction
4. **Design** scene composition (objects, lights, cameras, materials)
5. **Implement** with Three.js
6. **Add** interactions (orbit controls, click handlers, animations)
7. **Optimize** performance (instancing, LOD, frustum culling)
8. **Deliver** working 3D experience

## Notes
- Use `threejs` skill reference guides for implementation patterns
- Focus on performance — 60 FPS target
