description: Merge code from one branch to another
---

## Workflow

1. **Identify** source and target branches
2. **Check** for uncommitted changes (stash if needed)
3. **Checkout** target branch
4. **Merge** source branch
5. **Resolve** conflicts if any
6. **Verify** merge succeeded
7. **Report** result

## Notes
- Ask user to confirm before merging
- Handle merge conflicts interactively
- Never force-push without explicit approval
