## Purpose

Execute tasks using configured MCP servers.

## Workflow

1. **Check available MCP tools**: List tools from configured MCP servers
2. **Select appropriate tool**: Match the task to the best available MCP tool
3. **Execute**: Run the MCP tool with appropriate parameters
4. **Report**: Return results to the user

## Fallback

If no suitable MCP tool is found, report back and suggest alternative approaches.

## Notes

- MCP servers are configured via IDE settings
- Use the IDE's built-in MCP support for tool discovery and execution
